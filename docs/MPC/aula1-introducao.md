# Aula 1 - Introdução ao HTML

## 🧠 O que é HTML?

HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas web.

---

## 🧱 Estrutura básica

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Minha primeira página</title>
  </head>
  <body>
    <h1>Olá mundo!</h1>
    <p>Minha primeira página em HTML</p>
  </body>
</html>
