# 💻 ALP - Algoritmos e Lógica de Programação

Nesta disciplina você aprenderá a criar algoritmos e programas incríveis que podem resolver problemas reais. Vamos aprender a programar de forma divertida e interativa! 🚀

## 📌 Conteúdos:
- Lógica e raciocínio
- Formas de Representação de Algoritmos
- Tipos de Dados, Variáveis e Operadores Lógicos
- Estruturas de Controle Sequenciais
- Estruturas Condicionais
- Estruturas de Repetição
- Funções e Procedimentos
- Estruturas Homogêneas - Vetores e Matrizes

## 🎯 Objetivo
1. Entender o que são algoritmos e como eles funcionam.
2. Construir algoritmos utilizando técnicas de programação estruturada e modular.
3. Conhecer as principais estruturas utilizadas na programação.
4. Aprender a utilizar ferramentas de versionamento e controle.

## 🛠️ Ferramentas
- **VisualG:** Para você criar e testar seus algoritmos de maneira simples.
- **CodeBlocks:** Ferramenta utilizada para programar em C!
