# 📚 Materiais de Algoritmos e Lógica de Programação – FATEC 2026

Repositório com materiais didáticos da disciplina de Algoritmos e Lógica de Programação ministrada na FATEC.

Este projeto reúne conteúdos voltados ao ensino de programação, incluindo exemplos práticos, listas de exercícios e materiais de apoio utilizados em sala de aula.

## 🎯 Conteúdo

- 📄 Listas de exercícios em Portugol
- 💻 Exemplos de algoritmos em C
- 🧠 Conceitos de lógica de programação
- 🔁 Estruturas de controle (if, while, for)
- 📊 Vetores e matrizes
- 🧩 Funções e modularização

## 🚀 Objetivo

Auxiliar estudantes no aprendizado de programação, com foco em resolução de problemas e desenvolvimento do raciocínio lógico.

## 🛠️ Tecnologias utilizadas

- Linguagem C
- Portugol
- Git e GitHub

## 📌 Público-alvo

- Estudantes de tecnologia
- Iniciantes em programação
- Alunos da FATEC
